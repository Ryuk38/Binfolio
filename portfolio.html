Doctor Appointment Scheduling System


Led the development of a Doctor Appointment Scheduling System for a college project, utilizing VB.Net for the frontend and MySQL for the backend. This system streamlines healthcare facility operations by providing robust administrative tools for managing doctor profiles, scheduling patient appointments, and generating custom reports. Key features include secure admin authentication, efficient doctor management, user-friendly appointment scheduling, and insightful reporting capabilities. This project enhanced my skills in VB.Net and MySQL and emphasized the importance of user-centric design and data-driven decision-making in healthcare administration.

Hospital Management System


Developed a Hospital Management System as part of my class 12 computer science project, utilizing Python for the application logic and MySQL for the backend database. This system facilitates efficient handling of various hospital operations, including token generation, appointment scheduling, medical records management, and ambulance booking. The project showcases my ability to create comprehensive solutions for complex administrative tasks in a healthcare setting, emphasizing my skills in Python programming and database management.

Parking Slot Booking System


Led the development of a robust Parking Slot Booking System for malls using VB.Net for the frontend and MySQL for the backend. This system offers a user-friendly interface for real-time booking, ensuring efficient allocation and management of parking slots. Key features include secure user authentication, diverse slot registration (car, motorbike, electric vehicle), and comprehensive report generation. This project enhanced my skills in VB.Net, MySQL, and user-centric design, while deepening my understanding of resource allocation and data management in real-time systems.
